## Что такое блокчейн?  
Блокчейн — это цифровой реестр, в котором хранятся записи о транзакциях. Он распределен между участниками сети, что делает его прозрачным, безопасным и неизменяемым.  

### Преимущества блокчейна:  
1. **Прозрачность:** все участники сети видят историю транзакций.  
2. **Безопасность:** данные защищены криптографией.  
3. **Неизменяемость:** записи невозможно изменить без пересчета всех последующих блоков.

## Применение блокчейна в голосовании  
Система электронного голосования на блокчейне позволяет:  
- Обеспечить прозрачность процесса.  
- Исключить фальсификацию результатов.  
- Обеспечить безопасность данных.  

### Используемые технологии:  
- **Python:** язык программирования для создания системы.  
- **Blockchain:** основа для хранения данных.  
- **Cryptography:** защита информации.  
- **SHA-256:** криптографический хэш для проверки целостности данных.  
- **RSA:** шифрование и цифровые подписи для подтверждения подлинности.  

## Как работают ключи?  
Каждый пользователь имеет:  
1. **Закрытый ключ:** используется для подписей и доступа к данным.  
2. **Открытый ключ:** может быть передан другим, не раскрывая личной информации.  

С помощью закрытого ключа создается цифровая подпись, которая защищает данные и подтверждает авторство. Открытые ключи используются для передачи данных, обеспечивая их безопасность.  

### Рекомендации:  
- Не используйте один и тот же открытый ключ многократно, чтобы избежать отслеживания.  
- Обязательно защищайте закрытые ключи.  

## PyCrypto и безопасность  
Для обеспечения безопасности проекта важно использовать актуальные версии библиотек.  

### Проблемы старых версий PyCrypto:  
- **Уязвимость Crypto.Random**: небезопасное использование при форках (до версии 2.6.1).  
- **Проблемы с RandomPool**: небезопасность потоков и недостаток энтропии (до версии 2.1.0).  

Рекомендуется переходить на **PyCryptodome**, так как PyCrypto устарел.  

### Основные модули:  
| Модуль               | Описание                                                                 |
|----------------------|-----------------------------------------------------------------------|
| **Crypto.Hash**      | Хэш SHA-256 для проверки целостности.                                |
| **Crypto.PublicKey** | RSA для шифрования и цифровых подписей.                             |
| **Crypto.Signature** | Протокол цифровой подписи PKCS1_v1_5.                                |
